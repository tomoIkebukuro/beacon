### アプリの概要
現在地周辺の人と会話できるSNSアプリです。投稿したコメントを地図上で確認したり範囲を絞って投稿を検索することができます。

### なぜ作ったのか
私は幼少期より引っ越しや転校が多く地元との繋がりも薄かったので、住んでいる場所のことや地元の人との繋がりを持ちたいと思っていました。しかし、それを実現するアプリがまだ世に存在していなかったので自分で作ることにしました。
